# Pong em Pygame

Um jogo Pong simples desenvolvido em **Python + Pygame**, usado para demonstração sobre lógica de jogos, eventos, movimentação, colisões e loop de jogo.

---

## Funcionalidades

- Duas barras (jogador e CPU)
- Bola com movimento contínuo
- Colisões com barras e bordas
- Pontuação automática
- Reinício da bola após cada ponto
- FPS controlado com `pygame.time.Clock`
---

## Pré-requisitos

- Python 3.8+
- Pygame

Instalação:

```bash
pip install pygame
```
Como rodar
```bash
python pong.py
```
